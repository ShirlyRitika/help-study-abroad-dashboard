Overview
Frontend web application built using Next.js,Material UI, Zustand, and NextAuth, integrated with the DummyJSON API.

Features

* Authentication with NextAuth
* Protected routes (Dashboard, Users, Products, Profile)
* Users list with pagination & search
* Products list with pagination, search & category filter
* Single user & product pages
* Responsive UI using Material UI
* State management with Zustand + client-side caching

Setup & Run

npm install
npm run dev

http://localhost:3000
